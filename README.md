# ademola.dev 

Source code for my personal portfolio site. Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools.

## Pages

- **Home** (`index.html`) — Scattered floating nav layout with name and tagline
- **About** (`about.html`) — Background, story, and skill stack
- **Experience** (`experience.html`) — Timeline of work and open-source contributions
- **Projects** (`projects.html`) — Things I've built and shipped
- **Contact** (`contact.html`) — Ways to reach me

## Stack

HTML · CSS · Vanilla JS · Google Fonts (Space Grotesk, Caveat)
