Project 01 · Wikimedia Commons · Volunteer

Cat-a-lot Gadget

Improved the Cat-a-lot JavaScript gadget, a widely-used tool on Wikimedia Commons for bulk photo categorisation. Investigated and fixed a critical bug where selecting a protected file caused the in-progress dialog to hang indefinitely. Enhanced error handling with callbacks, created user-friendly dialogs (Ignore & continue / Stop editing), improved feedback to users, added dark mode support, and modularised the error display logic. The fix was reviewed and merged into the production version.
